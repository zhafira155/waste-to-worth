Iklan Bundling Matematika

Cara Penyelesaian Spltv dari Iklan Produk


Persamaan


  1. x + y + z=130.000
  2. x + 2y + z=185.000
  3. 2x + y + z=165.000

Penyelesaian:

  1. Eliminasi dari persamaan 2 dan 1: kurangkan persamaan 2 dengan persamaan 1: (x + 2y + z = 185.000) - (x + y + z = 130.000)= -y = 55.000
  2. Eliminasi dari persamaan 3 dan persamaan 1: kurangkan persamaan 3 dengan persamaan 1: (2x + y + z = 165.000) - (x + y + z = 130.000)= -x = 35.000
  3. Subsititusikan ke persamaan 1: x + y + z = 130.000, 35.000 + 55.000 + z = 130.000, 90.000 + z = 130.000, z= 130.000 - 90.000, z=40.000
  4. Himpunannya {(35.000, 55.000, 40.000)}

Jadi, harga lampu jamur(x) adalah 35.000 - harga rak susun (y) adalah 55.000 dan harga rak make up (z) adalah 40.000
