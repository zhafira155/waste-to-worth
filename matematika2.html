Iklan Bundling Matematika

Cara Penyelesaian Spltv dari Iklan Produk


Persamaan


  1. x + y + z=125.000
  2. 2x + y + z=165.000
  3. x + 2y + z=155.000

Penyelesaian:

  1. Eliminasi dari persamaan 2 dan 1: kurangkan persamaan 2 dengan persamaan 1: (2x + y + z = 165.000) - (x + y + z = 125.000)= x = 40.000
  2. Eliminasi dari persamaan 3 dan persamaan 1: kurangkan persamaan 3 dengan persamaan 1: (x + 2y + z = 155.000) - (x + y + z = 125.000)= y = 30.000
  3. Subsititusikan ke persamaan 1: x + y + z = 125.000, 40.000 + 30.000 + z = 125.000, 70.000 + z = 130.000, z= 130.000 - 70.000, z=55.000
  4. Himpunannya {(40.000, 30.000, 55.000)}

Jadi, harga rak make up(x) adalah 40.000 - harga jam dinding (y) adalah 30.000 dan harga rak susun (z) adalah 55.000
